# CodeAlpha_HangmanGame

## Description
This is a simple text-based Hangman game developed using Python.

## How it works
- The program selects a random word from a predefined list.
- The player has 6 attempts to guess the word.
- The user enters one letter at a time.
- If the guess is correct, the letter is revealed.
- If incorrect, attempts are reduced.
- The game ends when the word is guessed or attempts run out.

## Concepts Used
- Python loops (while loop)
- Conditional statements (if-else)
- Lists and strings
- Random module
